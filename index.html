The live editing CMS

Built on php & angularjs, cosmo is one of the first opensource, live editing, mobile first & responsive-minded content management systems.

Site Optimization

With cosmo, your site will auto minify, combine and cache all css and javascript on your site, allowing it to increase in speed with minimal code on the developers end.

Version Control

When you forget what changes have been made to a document, cosmo will help you remember by showing you the changes made, highlighting additions and deletions to help you see the history of your document.

Custom WYSIWYG

Our custom WYSIWYG editor, built with HTML5 allows the insertion of html blocks of codes (directives) along and quick insertion/editing of images, galleries, movies and more.

Live Editing

Editing the content on your website shouldn’t require menus, submenus and "previews" of your webpage, instead it should be a fluid and live editing experience. Cosmo allows you to not only edit text, but insert images, galleries, videos and even blocks of html code directly into your site.

Editable regions on your site can be controlled by the developer to have specific areas for editing to help control the design and continuity, or allow the user to input html visually into the site, allowing them to build out pages with custom layouts live on the site.

Responsive Images

Image handling shouldn’t be difficult or frustrating. It should just work, and continue to work when you hand it off to your client. Cosmo takes out the guess work of responsive images and allows you to upload images up to 5mb, will automatically resize the image into 4 sizes - 2048px (retina displays), 1024px, 512px and 320px and use the correct image for the correct device.

Web App

With AngularJS being at the core of cosmo, every installation becomes a single page web "app". This allows for faster page loads by only fetching the content that's been updated on a page, no more need to re-load things like html and javascript. Being a single page app also allows for transitions, allowing your page to fade in/out or slide in/out much like on mobile apps.

Prefetching also is built directly into the site. This allows the website to begin loading content that the user will be heading towards, to help load times and create less interruptions for the user

Version Control

Revisions to a document shouldn't be a guessing game. With cosmo you can quickly and easily tell who and when the document was edited, whether it's published along with the ability to see the exact deletions and additions made since the previous save.

If you're familiar with a github or other git based systems you'll be right at home, seeing deletions marked in red, and additions marked in green. Add new code to the page? Not a problem, the revision manager will mark any new additions or deletions of code to the page as well.

Insert Directives

No more need to remember shortcodes or any code at all. Now you can insert a block of interactive html within cosmo's WYSIWYG editor. Simply input the information needed and it will generate the HTML for you.

Directives can be set by the developer to feed blog posts, insert divs & even insert google maps.
